Sizes

Add size="lg", size="sm", or size="xs" for additional sizes.

size="lg" default size="sm" size="xs"

Block

Create block level buttons—those that span the full width of a parent—by adding block="block"

style="primary" size="lg" block="block" default block="block"

State

Create block level buttons—those that span the full width of a parent—by adding block="block"

state="active" state="disabled"
